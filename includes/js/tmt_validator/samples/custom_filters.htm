Custom filters

Filters are stored inside tmt_globalFilters, a global variable that contains a JavaScript object for filters. In order to add our own, we can call the tmt_filterInfo function. The tmt_filterInfo function requires 2 arguments

  1. Regular expression pattern for the content we want to remove
  2. The string that will replace the content (often just an empty string)

<script type="text/javascript">
tmt_globalFilters.a_to_f = tmt_filterInfo("[^a-fA-F]", "");
</script>

Custom Filter
Only letters, from a to f (both lower and uppercase)
