Conditional validation

Out of the box, the TMT Validator doesn't provide a way to perform conditional validation. This was a design choice. Creating a flexible and generic solution would require a lot of code and would undoubtedly fail to cover all contingencies.

Whenever we need to peform conditional validation, all we need to do is write a custom rule and use a custom attribute (see sample).

We then add tmt:required="conditional". This will instruct the validator to check for rules even if the field is currently empty.

In this example we want to validate a text field only if the associated checkbox is checked. We will first add a custom rule named "dependonbox". The XHTML code will look like this:

<input type="text" name="newmeal" id="newmeal" tmt:required="conditional" tmt:dependonbox="specialmeal" />

Now we need to write the JavaScript code for the custom rule, adding a method to the tmt_globalRules object. As always for custom rules, the method should expect the field's DOM node as its only argument and return a boolean:

// This will be triggered by the tmt:dependonbox attribute
tmt_globalRules.dependonbox = function(fieldNode){
 var relatedBox = document.getElementById(fieldNode.getAttribute("tmt:dependonbox"));
 // The field is required only if the box is checked
 var isRequired = relatedBox.checked;
 if(isRequired && fieldNode.value == ""){
  return false;
 }
 return true;
}

The sample below is basically the same as above, we just added a little bit of inline JavaScript code that enables/disables the related text field based on the checkbox's status. The full form code is:

<form action="index.htm" method="post" tmt:validate="true">
<fieldset>
<legend>Conditional validation (enabling/disabling field)</legend>
<table>
<tr>
<td><label>Yes, I am moving to a new job!
<input name="moving" type="checkbox" class="boxes" id="moving" onclick="document.getElementById('newjob').disabled=this.checked?false:true" value="true" />
</label>
</td>
</tr>
<tr>
<td>Specify the new job please
<input type="text" name="newjob" id="newjob" disabled="disabled" tmt:required="conditional" tmt:dependonbox="moving" tmt:message="Please enter the new job" />
<input type="submit" value="Submit" /></td>
</tr>
</table>
</fieldset>
</form>
